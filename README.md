# InAmigos Foundation Awareness Website

## Overview

The InAmigos Foundation Awareness Website is a responsive and informative web page developed using HTML and CSS. The purpose of this project is to spread awareness about the mission, initiatives, and social impact of InAmigos Foundation. The website highlights the organization's ongoing projects, achievements, and opportunities for individuals to contribute through volunteering and community participation.

---

## Features

* Responsive and modern user interface
* Fixed navigation bar with smooth scrolling
* Hero section with NGO-themed background and call-to-action buttons
* About section introducing InAmigos Foundation
* Projects section showcasing major NGO initiatives
* Gallery section displaying project activities and events
* Volunteer section encouraging community participation
* Contact section for user interaction
* Social media integration with clickable icons
* Consistent NGO-themed design and background overlays

---

## Technologies Used

* HTML5
* CSS3
* Font Awesome Icons
* Google Fonts (Poppins)

## NGO Initiatives Featured

### Project SEVA

Distributed over **50,000+ meals and clothing items** to underprivileged communities.

### Project BACHPANSHALA

Supports children's education through digital literacy, life skills training, and academic assistance.

### Project JEEV

Promotes animal welfare by feeding **50+ stray animals daily** through volunteer efforts.

### Project UDAAN

Empowers women through self-help groups, skill development, financial literacy, and menstrual hygiene awareness.

### Project PRAKRITI

Encourages environmental sustainability with **20,000+ saplings planted** and eco-friendly agricultural initiatives.

### Project VIKAS

Provides employment and skill development opportunities, having trained **30,000+ interns** across multiple professional domains.
